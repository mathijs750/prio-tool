# Project: Priority tasklist tool

## Project Overview

This tool is designed to make prioritising tasks easier for users. Task kan have 3 priority levels: A, B, and C. The tool will allow users to add tasks, assign priority levels, and view their task list sorted by priority. Task also have a complexity value: seed, plant or tree. The tool will nudge users to break down complex tasks into smaller, more manageable ones. The goal is to help users focus finish the small, high-priority tasks first, while also keeping an eye on the bigger picture.
